# SOMA Marketing System — Roadmap

## 🚀 START (Trojka — gradimo prvo)
| Agent | Fokus | Node pattern |
|-------|-------|-------------|
| Trend Intelligence | Jutarnji scan trendova po niši | search → filter → ranked lista |
| Hook Writer | Prvih 3s videa / prva rečenica posta | input → 5 opcija → rank |
| Content Repurposer | 1 ideja → 5 formata za 5 platformi | input → adapt → 5 verzija |

## 📊 Tier 2 — Optimizacija (kada trojka radi)
- **Performance Analyst** — brojevi → akcije, korelacije sadržaj/rezultati
- **YouTube SEO** — naslov, opis, tagovi, thumbnail brief
- **LinkedIn Thought Leadership** — tehničke ideje → human stories

## 🔥 Tier 3 — Skaliranje (kada imaš podatke)
- **Content Calendar** — 30-dnevni plan iz trendova + analytics
- **Competitor Intelligence** — gap analysis, šta konkurenti propuštaju
- **Comment & DM Analyzer** — pitanja publike → nove teme sadržaja

## Orchestracija
```
Trend Intelligence
      ↓ A2A
Hook Writer + Repurposer (parallel)
      ↓ A2A
Platform Writers: LinkedIn / TikTok / Instagram / YouTube
      ↑ feedback
Performance Analyst ←── Comment Analyzer
```

## Platforme
YouTube · Instagram · TikTok · LinkedIn · X

## SOMA pravila koja važe
- Svaki agent: max 3 noda, single responsibility
- Errors: LLM-judge <0.70 + self-check + human veto
- Instinct: situation→mistake→fix
